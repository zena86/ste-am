Адаптивность реализована за счет:
* медиазапросов
* гибкости верстки
* настраиваемой структуры макета страницы

Для компиляции SCSS (переменные, миксины, вложенность, палитра цветов и размеры шрифтов) использовалось приложение Koala v2.3.0.

* Info Windows
